NW.js port for ARM
============================= 

[NW.js](http://nwjs.io/ "NW.js web site") (formely node-webkit) binary compiled for the **ARMv7**.


## Versions
NW.js version: v0.26.3 

Chromium 62 + Node 9.0.0


## Tested on
* NXP i.MX6 Quad Processor with 2GB RAM using *IMXLinux BSP L4.1.15_2.1.0*
